# Research Papers

Papers to read during the learning phase.

## TEE Fundamentals

- [ ] Intel SGX Explained (Costan & Devadas, 2016)
  - https://eprint.iacr.org/2016/086.pdf
  - The definitive reference for understanding SGX architecture

- [ ] AMD SEV-SNP: Strengthening VM Isolation
  - https://www.amd.com/system/files/TechDocs/SEV-SNP-strengthening-vm-isolation-with-integrity-protection-and-more.pdf

## TEE Attacks

- [ ] Foreshadow: Extracting the Keys to the Intel SGX Kingdom (2018)
  - https://foreshadowattack.eu/
  - L1 terminal fault attack on SGX

- [ ] Plundervolt: Software-based Fault Injection Attacks against Intel SGX (2019)
  - https://plundervolt.com/
  - Voltage manipulation to corrupt enclave computation

- [ ] SGAxe: How SGX Fails in Practice (2020)
  - https://sgaxe.com/
  - Cache attack extracting attestation keys

## Custody & Key Management

- [ ] CHURP: Dynamic-Committee Proactive Secret Sharing (2019)
  - https://eprint.iacr.org/2019/017.pdf
  - Threshold secret sharing with committee changes

- [ ] FROST: Flexible Round-Optimized Schnorr Threshold Signatures (2020)
  - https://eprint.iacr.org/2020/852.pdf
  - Already studied — reference for integration

## TEE + Blockchain

- [ ] Teechain: A Secure Payment Network with Asynchronous Blockchain Access (2019)
  - https://arxiv.org/abs/1707.05454
  - Payment channels using TEEs

- [ ] Ekiden: A Platform for Confidentiality-Preserving, Trustworthy, and Performant Smart Contract Execution (2019)
  - https://arxiv.org/abs/1804.05141
  - TEE-based smart contract execution (now Oasis Network)

- [ ] Town Crier: An Authenticated Data Feed for Smart Contracts (2016)
  - https://eprint.iacr.org/2016/168.pdf
  - TEE oracle for blockchain

## Attestation

- [ ] Remote Attestation: Models and Protocols
  - Survey of attestation approaches

- [ ] DCAP (Data Center Attestation Primitives)
  - Intel's newer attestation model without Intel servers
